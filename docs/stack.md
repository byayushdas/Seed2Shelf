# Seed2Shelf Tech Stack

## Blockchain & Smart Contracts
- **Solidity ^0.8.20**: Writing the smart contracts for Ethereum.
- **Hardhat**: Development environment for compiling, testing, and deploying contracts.
- **OpenZeppelin Contracts**: Reusable, secure smart contract components (`Ownable.sol`).
- **ethers.js**: Library for interacting with the Ethereum Blockchain from the frontend.

## Frontend
- **Next.js 14 (Pages Router)**: Core React framework for routing and server-side rendering.
- **React**: UI library.
- **TypeScript**: Static typing for robust code.
- **Tailwind CSS**: Utility-first CSS framework customized with a deep green and gold agricultural theme.
- **lucide-react**: Lightweight SVG icons for the dashboards.
- **qrcode.react**: Library to dynamically generate SVG QR codes for consumer tracing.
